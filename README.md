# Getting and Cleaning Data Project

## Project Description

This project demonstrates cleaning and organizing data collected from the accelerometers of Samsung Galaxy S smartphones.

The goal was to create a tidy dataset that can be used for future analysis.

## Files

### run_analysis.R

This script:

1. Downloads the UCI HAR Dataset.
2. Loads training and testing datasets.
3. Combines the datasets.
4. Extracts mean and standard deviation measurements.
5. Uses descriptive activity labels.
6. Creates a final tidy dataset with averages by subject and activity.

### tidy_data.txt

The final cleaned dataset.

### CodeBook.md

Contains descriptions of variables and cleaning steps.
